Aquest directori conté tests d'unitat per a les diferents classes del projecte. Cada arxiu correspon a una classe específica del domini i conté tests que verifiquen el seu correcte funcionament.

Arxius:

BruteForceTest
Tests per a la classe BruteForce. Verifica que l'algoritme de BruteForce funciona correctament.

ProducteTest
Tests per a la classe Producte. Evalua que la gestió d'un producte es realitza correctament.
